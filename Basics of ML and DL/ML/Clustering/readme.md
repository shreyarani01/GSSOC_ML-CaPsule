# Clustering

This folder contains beginner-friendly implementations of popular clustering algorithms and related techniques used in unsupervised machine learning. Each notebook includes theory, implementation, visualization, and practical examples.

## Contents

### 1. K-Means Clustering

* Introduction to K-Means
* Working of the algorithm
* Data preprocessing
* Cluster visualization
* Performance analysis

### 2. Elbow Method

* Finding the optimal number of clusters
* WCSS (Within-Cluster Sum of Squares) analysis
* Elbow curve visualization

### 3. Silhouette Method

* Silhouette Score calculation
* Cluster quality evaluation
* Selecting the best value of **K**

### 4. Hierarchical Clustering

* Agglomerative clustering
* Cluster formation process
* Dendrogram visualization
* Choosing the optimal number of clusters

### 5. DBSCAN (Density-Based Clustering)

* Density-based clustering
* Core, Border, and Noise points
* Clustering arbitrary-shaped data
* Handling outliers effectively

### 6. K-Means vs DBSCAN

A comparison of both clustering algorithms, including:

* Working principle
* Advantages and limitations
* Performance comparison
* Suitable use cases
* Real-world applications
