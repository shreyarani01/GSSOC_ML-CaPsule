# 🛒 SmartCart Customer Clustering

## 📌 Overview

This project demonstrates **Customer Segmentation** using unsupervised machine learning techniques. The objective is to group customers with similar purchasing behavior so that businesses can better understand their customers and create targeted marketing strategies.

The project covers the complete clustering workflow, including data preprocessing, feature scaling, optimal cluster selection, model training, visualization, and cluster interpretation.

## 📊 Workflow

1. Load the dataset
2. Clean and preprocess the data
3. Perform Exploratory Data Analysis (EDA)
4. Scale numerical features
5. Determine the optimal number of clusters using the Elbow Method
6. Evaluate clustering using the Silhouette Score
7. Train the K-Means model
8. Visualize customer clusters
9. Analyze and interpret each customer segment
