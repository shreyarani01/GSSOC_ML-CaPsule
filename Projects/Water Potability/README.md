# Water_Potability_Prediction

## Dataset

- **Source:** Water Potability Dataset (Kaggle)
- The notebook uses the Water Potability dataset containing physicochemical properties of water samples for binary classification.
- **Dataset Link:** https://www.kaggle.com/datasets/devanshibavaria/water-potability-dataset-with-10-parameteres :contentReference[oaicite:0]{index=0}

This project predicts whether a water sample is **potable (safe to drink)** based on its physicochemical properties. It demonstrates an end-to-end machine learning workflow including data preprocessing, exploratory data analysis (EDA), feature engineering, model comparison, evaluation, and prediction.

## Key Features

1. Missing value handling and data preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature engineering and feature scaling
4. Comparison of multiple classification models
5. Model evaluation using Accuracy, Precision, Recall, F1-score, and ROC-AUC
6. Confusion Matrix and ROC Curve visualization
7. Permutation-based feature importance analysis
8. Water potability prediction for unseen samples

## Tech Stack

- Python
- Pandas, NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Machine Learning (Classification)

## Usage

1. Open `water_potability.ipynb` in Jupyter Notebook or VS Code.
2. Install the required Python libraries.
3. Run all notebook cells sequentially.
4. Refer to `Water_Potability_Report.pdf` for detailed methodology, visualizations, model comparison, and conclusions.
